# Clicker

A clicker written in C++ using the SDL2 library.

## Project structure

```bash
clicker
├── src
│   └── main.cpp
├── lib
│   ├── SDL2
│   ├── SDL2_image
│   └── SDL2_ttf
├── build
├── cmake
│   └── sdl2
│       ├── FindSDL2.cmake
│       ├── FindSDL2_image.cmake
│       └── FindSDL2_ttf.cmake
├── .gitignore
├── .gitmodules
├── CMakeLists.txt
└── README.md
```
